# memo cli

This is a memo app for terminal.

## Help

```
> memo -h
USAGE: memo <subcommand>

OPTIONS:
  -h, --help              Show help information.

SUBCOMMANDS:
  show
  list (default)
  insert
  remove
  drop
  s                       Alias for `show`
  l                       Alias for `list`
  i                       Alias for `insert`
  r                       Alias for `remove`
  d                       Alias for `drop`

  See 'memo help <subcommand>' for detailed help.
```
